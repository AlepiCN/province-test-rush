# 求R-增长率

## 考点1 一般增长率
R = X/A
根据题干信息直接求取

## 考点2 间隔增长率

### 一般间隔增长率
R = R1 + R2 + R1*R2
使用场景：2018-2019年增长率为R1，2019-2020年增长率为R2，求2018-2020年增长率R

### 逆运算
存在R及R1，求R2。此时，只需根据答案进行判断，R2略小于R-R1，因为需要减去R1*R2

### 间隔倍数
间隔倍数 = 间隔增长率 + 1

## 考点3 比值增长率
R=(R1-R2)/(1+R2)
使用场景：总销售额为M，增长率为R1；单价为N，增长率为R2。求销量增长率

## 考点4 成绩增长率
R = R1 + R2 + R1*R2
使用场景：单价为N，增长率为R1；销量为M，增长率为R2。求销售额增长率
特殊场景：总销售额为M，增长率为R1；品类占比为N，增长率为R2。求品类销售额增长率
