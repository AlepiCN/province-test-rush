## 📊 增长率相关 ⭐⭐⭐⭐⭐

### 基本公式

#### 增长率
```
增长率 = 增长量 ÷ 基期量
增长率 = (现期量 - 基期量) ÷ 基期量
增长率 = X/A （X为增长量，A为基期量）
```

#### 增长量
```
增长量 = 现期量 - 基期量
增长量 = 基期量 × 增长率
增长量 = 现期量 × 增长率 ÷ (1 + 增长率)
```

#### 基期量
```
基期量 = 现期量 ÷ (1 + 增长率)
基期量 = 现期量 - 增长量
```

#### 现期量
```
现期量 = 基期量 × (1 + 增长率)
现期量 = 基期量 + 增长量
```

### 特殊情况

#### 年均增长率
```
年均增长率 = (现期量 ÷ 基期量)^(1/n) - 1
其中n为年份数
```

#### 混合增长率
```
混合增长率 = (部分1增长率 × 权重1 + 部分2增长率 × 权重2) ÷ (权重1 + 权重2)
介于部分增长率之间，偏向权重大的部分
```

#### 间隔增长率
```
间隔增长率 = r1 + r2 + r1×r2
其中r1为第一年增长率，r2为第二年增长率
```

##### 间隔增长率逆运算
存在R及R1，求R2。此时，只需根据答案进行判断，R2略小于R-R1，因为需要减去R1×R2

##### 间隔倍数
```
间隔倍数 = 间隔增长率 + 1
```

#### 比值增长率
```
比值增长率 = (R1 - R2) ÷ (1 + R2)
```
使用场景：总销售额为M，增长率为R1；单价为N，增长率为R2。求销量增长率

#### 乘积增长率
```
乘积增长率 = R1 + R2 + R1×R2
```
使用场景：
- 单价为N，增长率为R1；销量为M，增长率为R2。求销售额增长率
- 总销售额为M，增长率为R1；品类占比为N，增长率为R2。求品类销售额增长率

## 📊 比重相关 ⭐⭐⭐⭐⭐

### 基本公式

#### 一般比重
```
比重 = 部分 ÷ 整体
比重 = 部分占比 × 100%
```

#### 部分
```
部分 = 整体 × 比重
部分 = 整体 × 比重%
```

#### 整体
```
整体 = 部分 ÷ 比重
整体 = 部分 ÷ 比重%
```

#### 易错点
```
1. 审题不清
2. 饼状图技巧：
   - 饼图顺时针顺序和题干描述顺序一致
   - 可以根据一个或两个小饼直接锁定答案
```

### 比重变化

#### 基期比重
```
基期比重 = 现期比重 × (1 + 整体增长率) ÷ (1 + 部分增长率)
基期比重 = 基期部分 ÷ 基期整体
```

#### 比重变化量
```
比重变化 = (部分增长率 - 整体增长率) ÷ (1 + 整体增长率)
比重变化 = 现期比重 - 基期比重
```

#### 判断比重升降
```
部分增长率 > 整体增长率 → 比重上升
部分增长率 < 整体增长率 → 比重下降
部分增长率 = 整体增长率 → 比重不变
```

#### 两期比重比较
```
比重趋势判断：
R部分 > R整体 ⇔ 比重上升
R部分 < R整体 ⇔ 比重下降
R部分 = R整体 ⇔ 比重不变

比重差计算：
比重差 = 现期比重 - 基期比重
       = (部分基期量 ÷ 整体现期量) × (部分增长率 - 整体增长率)
       = (a/B) × (R1 - R2)
       = (A/B) × (R1 - R2) ÷ (1 + R1)

快速判断技巧：
- a/B 一般小于1，所以 比重差 < |R1 - R2|
- 当a/B极小时，比重差必然极小
  例：a/B = 0.01，则比重差最多也只有1%
```

### 隔级比重

#### 隔级比重公式
```
当整体分为多个层级时使用

隔级比重(小比大) = (小集合 ÷ 中集合) × (中集合 ÷ 大集合)

隔级比重(小比中) = (小集合 ÷ 大集合) ÷ (中集合 ÷ 大集合)
```

#### 使用场景
```
例：求某省A市占全国GDP的比重
     全国GDP = 1000亿
     某省GDP = 100亿
     A市GDP = 10亿

解：隔级比重(A市占全国) = (A市GDP ÷ 某省GDP) × (某省GDP ÷ 全国GDP)
                         = (10 ÷ 100) × (100 ÷ 1000)
                         = 0.1 × 0.1
                         = 0.01 = 1%
```

## 📊 平均量相关 ⭐⭐⭐⭐

### 基本公式

#### 平均量
```
平均量 = 总量 ÷ 数量
平均量 = 总和 ÷ 个数
```

#### 总量
```
总量 = 平均量 × 数量
```

#### 数量
```
数量 = 总量 ÷ 平均量
```

### 平均量变化

#### 平均量增长率
```
平均量增长率 = (总量增长率 - 数量增长率) ÷ (1 + 数量增长率)
```

#### 基期平均量
```
基期平均量 = 现期平均量 × (1 + 数量增长率) ÷ (1 + 总量增长率)
```

## 📊 倍数相关 ⭐⭐⭐⭐

### 基本公式

#### 倍数
```
倍数 = A ÷ B
```

#### A比B多几倍
```
A比B多几倍 = (A - B) ÷ B = A÷B - 1
```

#### A是B的几倍
```
A是B的几倍 = A ÷ B
```

## 📊 综合公式 ⭐⭐⭐

### 间隔计算

#### 间隔基期量
```
间隔基期量 = 现期量 ÷ [(1 + r1) × (1 + r2)]
```

#### 间隔增长量
```
间隔增长量 = 现期量 - 间隔基期量
```

### 拉动增长

#### 拉动增长率
```
拉动增长率 = 部分增长量 ÷ 整体基期量
```

#### 贡献率
```
贡献率 = 部分增长量 ÷ 整体增长量
```

## 📊 混合溶液问题 ⭐⭐⭐⭐

### 基本公式

#### 混合溶液浓度
```
混合溶液浓度 R = (A×R1 + B×R2) ÷ (A + B)
```
其中：
- A：第一种溶液的质量
- B：第二种溶液的质量
- R1：第一种溶液的浓度
- R2：第二种溶液的浓度
- R：混合后的浓度

### 解题方法

#### 方法1：十字交叉法
```
适用条件：已知3个浓度，求两种溶液的质量比

公式：
A/B = (R - R2) ÷ (R1 - R)

使用步骤：
1. 画出十字交叉图
2. 将高浓度R1写在左上，低浓度R2写在左下
3. 将混合浓度R写在中间
4. 计算对角线差值：|R1 - R| 和 |R - R2|
5. 质量比 A/B = |R - R2| ÷ |R1 - R|

例：将10%和20%的盐水混合成15%的盐水，求质量比
解：A/B = (15% - 10%) ÷ (20% - 15%) = 5% ÷ 5% = 1
   即10%和20%的盐水质量比为1:1
```

#### 方法2：线段法
```
适用条件：已知2个浓度和1个质量比，求混合浓度

基本原理：
- 低浓度 < 混合浓度 < 高浓度
- 溶液越多，混合浓度离该溶液浓度越近
- 质量的比 = 浓度差的反比

公式：
m高/m低 = (R混 - R低) ÷ (R高 - R混)

即：质量比 = 浓度差的反比

例：将质量比为3:1的10%和20%的盐水混合，求混合浓度
解：m高/m低 = 3/1 = (R混 - 10%) ÷ (20% - R混)
   3 × (20% - R混) = 1 × (R混 - 10%)
   60% - 3R混 = R混 - 10%
   4R混 = 70%
   R混 = 17.5%
```

### 考点总结

#### 默认条件
```
盐水问题的默认条件：低浓度 < 混合浓度 < 高浓度
```

#### 题型1：已知3个浓度，求质量比
```
方法：用十字交叉法求解
```

#### 题型2：已知2个浓度和1个质量比，求混合浓度
```
方法：用线段法求解
```

#### 题型3：混合溶液的变体
```
增长率的溶液：基期量
各种平均值的溶液：看均到了什么东西上
```

### 易错点（坑点）

#### 坑点1：寻找溶液
```
问题：题目中可能隐藏溶液信息
解决：仔细阅读题干，找出溶液（如盐水问题中的盐和水）
```

#### 坑点2：浓度理解错误
```
问题：混淆浓度、质量、溶液的概念
解决：明确浓度是溶质质量÷溶液质量
```

#### 坑点3：质量比与浓度比混淆
```
问题：将质量比误认为浓度比
解决：记住质量比 = 浓度差的反比
```

## 💡 记忆口诀

```
增长率量基期量，三个公式要记牢
增长量是现减基，增长率是量除基

间隔增长要相加，再加乘积别忘啦
间隔倍数加一，逆运算看选项差

比值增长有技巧，分子减分母除一加
销量增长常用到，记准公式错不了

乘积增长和间隔，公式相同要分辨
单价销量销售额，占比增长也适用

比重部分除整体，部分整体乘比重
比重变化看增长率，部分大则比重升
基期比重现乘比，整体部分增长率
隔级比重分层级，小中大小依次乘
饼图顺时针顺序，快速锁定答案行

混合溶液十字叉，线段法里找浓度
质量比是浓度差，反比关系要记清
低混高浓度有序，溶液越多越靠近

平均量是总量除数量，平均增长率量减数
倍数就是A除B，多几倍要减一
年均增长开n次方，间隔增长加乘积
```
