## 📊 增长率相关 ⭐⭐⭐⭐⭐

### 基本公式

#### 增长率
```
增长率 = 增长量 ÷ 基期量
增长率 = (现期量 - 基期量) ÷ 基期量
增长率 = X/A （X为增长量，A为基期量）
```

#### 增长量
```
增长量 = 现期量 - 基期量
增长量 = 基期量 × 增长率
增长量 = 现期量 × 增长率 ÷ (1 + 增长率)
```

#### 基期量
```
基期量 = 现期量 ÷ (1 + 增长率)
基期量 = 现期量 - 增长量
```

#### 现期量
```
现期量 = 基期量 × (1 + 增长率)
现期量 = 基期量 + 增长量
```

### 特殊情况

#### 年均增长率
```
年均增长率 = (现期量 ÷ 基期量)^(1/n) - 1
其中n为年份数
```

#### 混合增长率
```
混合增长率 = (部分1增长率 × 权重1 + 部分2增长率 × 权重2) ÷ (权重1 + 权重2)
介于部分增长率之间，偏向权重大的部分
```

#### 间隔增长率
```
间隔增长率 = r1 + r2 + r1×r2
其中r1为第一年增长率，r2为第二年增长率
```

##### 间隔增长率逆运算
存在R及R1，求R2。此时，只需根据答案进行判断，R2略小于R-R1，因为需要减去R1×R2

##### 间隔倍数
```
间隔倍数 = 间隔增长率 + 1
```

#### 比值增长率
```
比值增长率 = (R1 - R2) ÷ (1 + R2)
```
使用场景：总销售额为M，增长率为R1；单价为N，增长率为R2。求销量增长率

#### 乘积增长率
```
乘积增长率 = R1 + R2 + R1×R2
```
使用场景：
- 单价为N，增长率为R1；销量为M，增长率为R2。求销售额增长率
- 总销售额为M，增长率为R1；品类占比为N，增长率为R2。求品类销售额增长率

## 📊 比重相关 ⭐⭐⭐⭐⭐

### 基本公式

#### 一般比重
```
比重 = 部分 ÷ 整体
比重 = 部分占比 × 100%
```

#### 部分
```
部分 = 整体 × 比重
部分 = 整体 × 比重%
```

#### 整体
```
整体 = 部分 ÷ 比重
整体 = 部分 ÷ 比重%
```

#### 易错点
```
1. 审题不清
2. 饼状图技巧：
   - 饼图顺时针顺序和题干描述顺序一致
   - 可以根据一个或两个小饼直接锁定答案
```

### 比重变化

#### 基期比重
```
基期比重 = 现期比重 × (1 + 整体增长率) ÷ (1 + 部分增长率)
基期比重 = 基期部分 ÷ 基期整体
```

#### 比重变化量
```
比重变化 = (部分增长率 - 整体增长率) ÷ (1 + 整体增长率)
比重变化 = 现期比重 - 基期比重
```

#### 判断比重升降
```
部分增长率 > 整体增长率 → 比重上升
部分增长率 < 整体增长率 → 比重下降
部分增长率 = 整体增长率 → 比重不变
```

#### 两期比重比较
```
比重趋势判断：
R部分 > R整体 ⇔ 比重上升
R部分 < R整体 ⇔ 比重下降
R部分 = R整体 ⇔ 比重不变

比重差计算：
比重差 = 现期比重 - 基期比重
       = (部分基期量 ÷ 整体现期量) × (部分增长率 - 整体增长率)
       = (a/B) × (R1 - R2)
       = (A/B) × (R1 - R2) ÷ (1 + R1)

快速判断技巧：
- a/B 一般小于1，所以 比重差 < |R1 - R2|
- 当a/B极小时，比重差必然极小
  例：a/B = 0.01，则比重差最多也只有1%
```

### 隔级比重

#### 隔级比重公式
```
当整体分为多个层级时使用

隔级比重(小比大) = (小集合 ÷ 中集合) × (中集合 ÷ 大集合)

隔级比重(小比中) = (小集合 ÷ 大集合) ÷ (中集合 ÷ 大集合)
```

#### 使用场景
```
例：求某省A市占全国GDP的比重
     全国GDP = 1000亿
     某省GDP = 100亿
     A市GDP = 10亿

解：隔级比重(A市占全国) = (A市GDP ÷ 某省GDP) × (某省GDP ÷ 全国GDP)
                         = (10 ÷ 100) × (100 ÷ 1000)
                         = 0.1 × 0.1
                         = 0.01 = 1%
```

## 📊 平均量相关 ⭐⭐⭐⭐

### 基本公式

#### 平均量
```
平均量 = 总量 ÷ 数量
平均量 = 总和 ÷ 个数
```

#### 总量
```
总量 = 平均量 × 数量
```

#### 数量
```
数量 = 总量 ÷ 平均量
```

### 平均量变化

#### 平均量增长率
```
平均量增长率 = (总量增长率 - 数量增长率) ÷ (1 + 数量增长率)
```

#### 基期平均量
```
基期平均量 = 现期平均量 × (1 + 数量增长率) ÷ (1 + 总量增长率)
```

## 📊 倍数相关 ⭐⭐⭐⭐

### 基本公式

#### 倍数
```
倍数 = A ÷ B
```

#### A比B多几倍
```
A比B多几倍 = (A - B) ÷ B = A÷B - 1
```

#### A是B的几倍
```
A是B的几倍 = A ÷ B
```

## 📊 综合公式 ⭐⭐⭐

### 间隔计算

#### 间隔基期量
```
间隔基期量 = 现期量 ÷ [(1 + r1) × (1 + r2)]
```

#### 间隔增长量
```
间隔增长量 = 现期量 - 间隔基期量
```

### 拉动增长

#### 拉动增长率
```
拉动增长率 = 部分增长量 ÷ 整体基期量
```

#### 贡献率
```
贡献率 = 部分增长量 ÷ 整体增长量
```

### 常用计算技巧

#### 增长量速算
```
增长量 ≈ 现期量 × 增长率（当增长率<10%时）
```

#### 比重变化速算
```
比重变化 ≈ 部分增长率 - 整体增长率（当增长率较小时）
```
